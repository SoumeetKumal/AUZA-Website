INTRODUCTION

These guidelines ensure consistency across all brand touchpoints, helping us maintain a strong, recognizable presence while empowering our team and partners to represent AUZA authentically.

How to Use This Guide

This document serves as your reference for all brand-related decisions. From visual assets to tone of voice, these guidelines help maintain brand integrity while allowing creative flexibility where appropriate.

Core Question

When in doubt, ask: "Does this reflect AUZA's values of functional, accessible, and innovative design?"

Color Philosophy

Dark backgrounds create premium, modern aesthetic. Reserve light backgrounds for specific use cases. Red is strategic, not decorative.

Accessibility

Always maintain minimum 4.5:1 contrast ratio (WCAG AA). Dark backgrounds + Off-White text = 19:1 (excellent).

MOTION & ANIMATION

Principles

Purposeful

Every animation serves a clear function. No decoration.

Subtle

Swift, concise transitions that feel intuitive.

Performant

60fps minimum. Use transform and opacity.

Duration Standards

Duration Use Case
150ms Micro-interactions (hovers, toggles)
250ms State changes, short transitions
400ms Complex transitions, modals

Easing Functions

Function Usage
ease-in-out Standard UI movements
ease-out Elements entering
ease-in Elements exiting

Accessibility

Always respect prefers-reduced-motion. When enabled, reduce animations to instant opacity changes.

PATTERNS & TEXTURES

Pattern Library

Dark Grid

Subtle red grid on dark. 5-10% opacity.

Dot Matrix

Minimal dot pattern with red accent.

Diagonal Lines

Dynamic diagonal suggesting progress.

Usage Guidelines

Always use at 5-15% opacity to avoid overwhelming content
Ensure sufficient contrast with overlaid content
Use sparingly - 1-2 patterns maximum per page
Best for large background areas or section dividers
